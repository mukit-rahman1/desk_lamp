# [W.I.P.] DeskLux Acrylic Desklamp
## Built on ESP32S3-Devkitc  

## Features
- 3 OLED modules on I2C with a TCA9548A MUX
- 2 LED strip modes: Progressive and Static
- 2 Push buttons to switch between modes

 ## Status
 - PCB is missing resistors
 - Acrylic is still arriving

> NOTE: Only had one TCA9548A MUX and it is soldered on to the PCB. The current main.ino has testing code without the MUX and supports 2 OLED modules.
